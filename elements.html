Elements Website

About

Website sharing information on five elements that I have chosen which are Helium, Hydrogen, Lithium, Boron and Oxygen. The website is aimed at students so I chose to include both detailed explanations and visual elements such as video.

I chose to include video examples of each element as I think that it is always easier to remember something when you can see a visual image of it.

Animations

I included some animations as-well such as the homepage which has an animated introductory video created in After Effects and I also included animated 'atoms' for each element to give it a bit more visual appeal.
